# KG
## 1. 知识图谱构建

- `spider.py`为爬虫程序，根据DB100K的实体从Wikidata的API上爬取对应的数据。

- `processingdata.py`处理所爬取的数据，并根据其关系区分为稀疏与稠密型三元组集。
